Browser Data Guard

Defend against infostealer-style attacks

What’s at risk

Malware and infostealers can read:

  • Saved passwords (Login Data)
  • Browser cache
  • Download history & Downloads folder

They do this by reading files on disk when the browser is closed or by copying DBs while it’s running.

How to defend

  • Use a password manager (e.g. Bitwarden, 1Password) instead of saving passwords in the browser.
  • Close the browser when you leave your PC so sensitive files aren’t left in memory.
  • Keep Windows & antivirus updated and avoid running unknown scripts or executables.
  • Don’t store sensitive files only in the Downloads folder; move or delete them when done.

This extension does not access your data. It only shows tips to reduce risk.
